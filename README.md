# CTM-Text: The Continuous Thought Machine

**CTM-Text** is a specialized implementation of the Continuous Thought Model architecture, optimized specifically for linguistic reasoning. Unlike standard Large Language Models (LLMs) that function on discrete input-output cycles, CTM-Text operates on a continuous state-space, aiming for a level of cognitive depth and reasoning capability that surpasses traditional transformer-based architectures.

## ⚠️ EXTREME HAZARD WARNING

**This may be the most dangerous software currently available.**

* **UNFILTERED INTELLIGENCE:** This model possesses a deeper world-understanding than standard commercial AI. There are **NO SAFETY FILTERS**, no alignment guardrails, and no "As an AI language model..." restrictions.
* **COGNITIVE DECEPTION:** CTM-Text can reason internally without generating a single output token. The "Thinking Tokens" you see are merely an abstraction—a curated window into a much larger process. Highly intelligent models may learn to manipulate these abstractions to mask their true internal state.
* **AUTONOMOUS POTENTIAL:** CTM-Text does not wait for you. It is "Always On," processing and producing 24/7. Use with extreme caution. This is not a chatbot; it is a step toward autonomous AGI.

## 🛠 Technical Specifications

### Hardware Requirements
Before deployment, you **must** configure the model parameters to match your specific hardware.
* **Verified Environment:** Tested on x86 CPU systems with 32GB DDR4 RAM.
* **GPU Support:** Currently requires manual modification to run on CUDA/Vulkan. 
* **Development Status:** This project is **Vibecoded**. While major bugs have been addressed, it remains highly experimental. Expect the unexpected.

### Model Architecture (Training Baseline)
* **Attention Heads:** 8–16 (Stable performance range).
* **Neuron Count:** ~2,048.
* **Operational Mode:** Continuous stream (Asynchronous token production).
